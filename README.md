# Barcode
Sistema WEB de Código de Barras para automatizar el control y gestión del envío y recepción de los bultos y paquetes entre las sucursales para una empresa de encomiendas, desarrollado en Python- Django y PostgreSQL funcionando en Servidor WEB Linux, el sistema cuenta con varios módulos, uno para la carga de información de los clientes, exportación de reportes sobre los bultos y paquetes en PDF, impresión de etiquetas de código de barras en impresora Zebra GX420D generadas en WORD, un módulo de escaneo de las etiquetas con una lectora inalámbrica, aviso por correo electrónico a los Clientes sobre el estatus de sus paquetes/envíos, un módulo donde los Clientes pueden consultar de manera on-line.
